# Django Views

Este projeto é um exercício desenvolvido durante o curso da EBAC para praticar a criação e uso de **views no Django**.

## Objetivo

Aprender e demonstrar a utilização de views no framework Django, incluindo:

- Criação de views baseadas em função (FBV - Function Based Views)
- Criação de views baseadas em classe (CBV - Class Based Views)
- Uso de URLs para mapear as views
- Renderização de templates HTML a partir das views
